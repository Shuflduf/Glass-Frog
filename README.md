# Glass Frog Academy

Glass Frog Academy (GFA) is a platform designed to support students in their academic journey, particularly focusing on the crucial transition from middle school to high school. Our mission is to prepare students to thrive in high school and beyond through engaging online classes, in-person hackathons, and interactive events.

## 🐸 What is "Glass Frog"?
The name "Glass Frog" embodies our philosophy:
- **Glass:** Represents clarity and a window into the future.
- **Frog:** Symbolizes the academic leap students make as they transition to high school.

## 🎯 Our Mission
We aim to equip students with the skills and tools needed to navigate complex challenges, think critically, and embrace their potential. With a focus on the International Baccalaureate (IB) curriculum, we offer tailored educational resources and experiences to make learning fun, engaging, and effective.

## 📚 Features
- **Summer Online Courses**: Tailored classes in IB Science, Math, Writing, and advanced topics like Physics, Biology, and Computer Science.
- **In-Person Events**: Competitions and hackathons to build skills through interactive activities.
- **Future Readiness**: Guidance and experience from current IB high school students to help younger students adapt to the high school workflow.

## 🛠️ Technology Stack
This project is built using:
- **Astro**: A modern static site generator.
- **Tailwind CSS**: For responsive and utility-first styling.
- **TypeScript & JavaScript**: For dynamic functionality.
- **Radix UI**: For custom UI components such as scroll areas and buttons.

## 🚀 Installation
To set up this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Shuflduf/Glass-Frog.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Glass-Frog
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:4321`.
